# Gesture Control AI (Virtual Mouse)

這是一個基於 Python 與電腦視覺的虛擬滑鼠控制程式，透過攝影機捕捉手部動作來控制電腦操作。專案使用了 OpenCV 進行影像處理，並結合手部關鍵點偵測來辨識不同的手勢，進而觸發滑鼠移動、點擊與滾動等指令。

## 功能與手勢說明

程式啟動後，請將手移入鏡頭內。以下為支援的操作手勢：

*   **移動滑鼠**：只有大拇指與食指伸出（指槍手勢 🔫）
*   **點擊 / 拖曳**：大拇指與食指捏合（Pinch）
*   **頁面滾動**：食指與中指朝上（比 YA ✌️）並上下移動
*   **按下右 Alt 鍵**：只有小拇指朝上（打勾勾手勢）
*   **退出程式**：按下鍵盤 `ESC` 鍵

## 環境需求

請確認已安裝 Python 環境，並安裝所需的依賴套件。你可以透過 `requirements.txt` 來安裝（若有提供）：

```bash
pip install -r requirements.txt
```

主要依賴套件：
*   `opencv-python`
*   `mediapipe` (用於手部追蹤)
*   `pyautogui` (用於控制滑鼠與鍵盤)

## 如何執行

在專案目錄下執行以下指令啟動程式：

```bash
python main.py
```

## 注意事項

*   請確保你的環境光線充足，且手部完全進入攝影機可視範圍內，以獲得最佳的辨識效果。
*   移動滑鼠時，程式會在畫面上畫出一個紫色框，手部在框內的移動會對應到整個螢幕，這有助於避免手部需要移動過大範圍。
